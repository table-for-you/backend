# Table-For-You

식당 예약 관리 서비스 서버입니다.

## ✨ Features
### 권한별
 - 관리자
     - 사용자 관리
     - 식당 등록 승인 및 관리
 - 식당 주인
     - 식당 생성
     - 예약자 관리
 - 사용자
     - 식당 예약 (번호표 & 시간)
     - 리뷰 생성
     - 방문한 식당 관리
     - 관심 식당 관리
  
### 공통
 - 회원가입
 - 로그인
 - 로그아웃
 - 알림 (FCM)

## 🚀 Tech Stacks
 - Language: Java
 - Build tool: Gradle
 - Frameworks & Libraries: Spring Boot, Spring Data JPA, Spring Security, Lombok, JJWT, QueryDSL, Redis(Redisson), firebase
 - Databases & Caching: MySQL, Redis
 - DevOps & Cloud: GitHub Actions, Docker, AWS
